# KrishnaBot — AI Chatbot 🤖

## Overview
An AI-powered chatbot with memory, personality and 
professional features built using Python and Llama AI.

## Features
✅ Remembers full conversation history
✅ Custom personality via system prompt
✅ Error handling — never crashes
✅ Commands: help, history, clear, quit
✅ Token optimization with MAX_HISTORY
✅ Powered by Meta's Llama 3.3 70B model

## Technologies Used
- Python
- Groq API
- Llama 3.3 70B (Meta AI)
- Google Colab

## How It Works
1. User sends message
2. Full chat history sent to AI API
3. AI responds with context of full conversation
4. Response added to history
5. Repeat

## Key Concepts Implemented
- LLM API integration
- Prompt Engineering
- Chat history management
- Error handling
- Modular code structure

## How to Run
1. Get free API key from console.groq.com
2. Open krishnabot.ipynb in Google Colab
3. Replace YOUR_KEY_HERE with your API key
4. Run all cells
5. Start chatting!
